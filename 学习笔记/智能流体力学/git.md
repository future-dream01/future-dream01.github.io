## git使用方法
1. 本地署名：
   -  `git config --global user.name "future-dream01"`
   -  `git config --global user.email "1561404532@qq.com"`
2. 克隆：`git clone 仓库链接`
3. 切换分支：
   - 切换到项目根目录：`cd 项目根目录`
   - 切换分支：`git checkout 分支名` 
4. 提交与推送：
   - 查看当前改了哪些文件：`git status`
   - 把修改过的文件加入暂存：`git add .` 
   - 提交：`git commit -m "修改的大致描述"`
   - 拉取最新的代码，解决可能的冲突：`git pull`，第一次拉取需要加上`origin 分支名`
   - 推送：`git push`，第一次推送需要加上`origin 分支名`
