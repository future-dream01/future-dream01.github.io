## git使用方法
1. 克隆：`git clone 仓库链接`
2. 切换分支：
   - 切换到项目根目录：`cd 项目根目录`
   - 切换分支：`git checkout 分支名` 
3. 提交与推送：
   - 查看当前改了哪些文件：`git status`
   - 把修改过的文件加入暂存：`git add .` 
   - 提交：`git commit -m "修改的大致描述"`
   - 拉取最新的代码，解决可能的冲突：`git pull`，第一次拉取需要加上`origin 分支名`
   - 推送：`git push`，第一次推送需要加上`origin 分支名`
